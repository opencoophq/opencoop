Ce guide vous accompagne dans la configuration de votre coopérative sur OpenCoop.

## 1. Créez votre compte

Rendez-vous sur [opencoop.be](https://opencoop.be) et cliquez sur **Commencer**. Choisissez un plan adapté à votre coopérative et complétez le formulaire d'inscription :

- **Nom de la coopérative** — Le nom officiel de votre coopérative
- **Slug** — Un identifiant compatible URL (ex. `ma-coop`) utilisé dans l'URL de votre page publique
- **Vos coordonnées** — Nom et adresse e-mail pour le compte administrateur

Vous recevrez un e-mail de confirmation pour vérifier votre compte.

## 2. Configurez votre coopérative

Une fois connecté, accédez aux **Paramètres** dans la barre latérale d'administration pour configurer :

- **Image de marque** — Téléchargez votre logo et définissez les couleurs de votre coopérative
- **Classes de parts** — Définissez une ou plusieurs classes de parts avec des prix et des règles
- **Paramètres e-mail** — Activez les e-mails automatiques pour les confirmations de transactions et les notifications aux actionnaires
- **Compte bancaire** — Ajoutez l'IBAN de votre coopérative pour le rapprochement des paiements

## 3. Ajoutez des actionnaires

Allez dans **Actionnaires** et cliquez sur **Ajouter un actionnaire**. Vous pouvez enregistrer trois types :

- **Particulier** — Une personne physique
- **Entreprise** — Une personne morale (numéro KBO/BCE requis)
- **Mineur** — Une personne de moins de 18 ans (représentant légal requis)

## 4. Traitez les transactions

Naviguez vers **Transactions** pour enregistrer les achats, ventes et transferts de parts. Chaque transaction génère :

- Un **code OGM** unique (communication structurée) pour le suivi des paiements
- Une notification automatique par e-mail à l'actionnaire (si activé)
- Un document de confirmation en PDF
