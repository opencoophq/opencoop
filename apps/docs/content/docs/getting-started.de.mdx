Dieser Leitfaden führt Sie durch die Einrichtung Ihrer Genossenschaft auf OpenCoop.

## 1. Erstellen Sie Ihr Konto

Besuchen Sie [opencoop.be](https://opencoop.be) und klicken Sie auf **Loslegen**. Wählen Sie einen Plan, der zu Ihrer Genossenschaft passt, und füllen Sie das Onboarding-Formular aus:

- **Name der Genossenschaft** — Der offizielle Name Ihrer Genossenschaft
- **Slug** — Eine URL-freundliche Kennung (z.B. `meine-geno`), die in der URL Ihrer öffentlichen Seite verwendet wird
- **Ihre Daten** — Name und E-Mail-Adresse für das Administratorkonto

Sie erhalten eine Bestätigungs-E-Mail zur Verifizierung Ihres Kontos.

## 2. Konfigurieren Sie Ihre Genossenschaft

Nach der Anmeldung navigieren Sie zu **Einstellungen** in der Admin-Seitenleiste, um Folgendes zu konfigurieren:

- **Branding** — Laden Sie Ihr Logo hoch und legen Sie die Farben Ihrer Genossenschaft fest
- **Anteilsklassen** — Definieren Sie eine oder mehrere Anteilsklassen mit unterschiedlichen Preisen und Regeln
- **E-Mail-Einstellungen** — Aktivieren Sie automatische E-Mails für Transaktionsbestätigungen und Benachrichtigungen an Anteilseigner
- **Bankkonto** — Fügen Sie die IBAN Ihrer Genossenschaft für den Zahlungsabgleich hinzu

## 3. Fügen Sie Anteilseigner hinzu

Gehen Sie zu **Anteilseigner** und klicken Sie auf **Anteilseigner hinzufügen**. Sie können drei Typen registrieren:

- **Einzelperson** — Eine natürliche Person
- **Unternehmen** — Eine juristische Person (KBO/BCE-Nummer erforderlich)
- **Minderjähriger** — Eine Person unter 18 Jahren (gesetzlicher Vertreter erforderlich)

## 4. Verarbeiten Sie Transaktionen

Navigieren Sie zu **Transaktionen**, um Käufe, Verkäufe und Übertragungen von Anteilen zu erfassen. Jede Transaktion erzeugt:

- Einen einzigartigen **OGM-Code** (strukturierte Mitteilung) zur Zahlungsverfolgung
- Eine automatische E-Mail-Benachrichtigung an den Anteilseigner (falls aktiviert)
- Ein PDF-Bestätigungsdokument
